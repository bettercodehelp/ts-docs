# flip

The flip module simulates a coin flip and returns a `1` for heads or a `0` for tails.

## Example

- `1` : heads
- `0` : tails

```ts
// Import the module
import { flip } from "easyscript.ts";

// Generate the result
const result  = flip();

// Log result to console
console.log(result);
// 0 or 1
```
